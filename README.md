# AI Internship

I am currently doing a **2-month internship at MirAI School of Technology**, where I am working on hands-on AI projects and learning how to build practical applications using modern tools and frameworks. This internship is helping me strengthen my skills in **Python, AI development, chatbot design, and interactive web app building**.

During this internship, I am completing assignments that combine creativity, problem-solving, and real-world implementation. Each project is designed to improve my understanding of how AI-powered applications are created and deployed.

## Assignment 1: Identity-Echo-Interface

The first assignment is **Identity-Echo-Interface**, built using **Python** and **Streamlit**. In this app, I collect a user's **name** and **message**, validate the input, display a personalized success response, and estimate the message's token usage based on its character count. This project demonstrates the basics of building a dynamic and user-friendly web application.

## Assignment 2: AI Multiverse Chatbot

The second assignment is **AI Multiverse Chatbot**, where users can chat with multiple personalities using **Streamlit** and the **Gemini API**. This project allows different conversational styles or personalities to be explored within a single chatbot experience, making it more interactive and interesting.
